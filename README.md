# 股票数据可视化工具

基于腾讯股票接口的实时行情可视化应用。

## 功能特点

- 📊 实时显示涨跌家数、涨跌幅分布图表
- 📈 支持单只股票查询
- 🔄 自动/手动刷新数据
- 📉 市值分布统计

## 快速开始

### 1. 安装依赖

```bash
pip install -r requirements.txt
```

### 2. 运行程序

```bash
python main.py
```

## 文件说明

| 文件 | 说明 |
|------|------|
| `main.py` | 主程序（GUI界面） |
| `stock_api.py` | 腾讯股票接口封装 |
| `requirements.txt` | Python依赖 |

## 接口说明

使用腾讯财经实时行情接口：
```
https://qt.gtimg.cn/q=股票代码
```

支持的股票代码格式：
- 深圳: `sz000858` 或直接 `000858`
- 上海: `sh600519` 或直接 `600519`
- 北交所: `bj430047`

## 截图预览

运行后将显示类似以下界面：
- 左侧：多个实时图表（涨跌家数、涨跌幅分布等）
- 右侧：控制面板（股票查询、数据刷新）

## 注意事项

1. 需要网络连接访问腾讯接口
2. 建议刷新间隔 ≥ 3秒，避免请求过于频繁
3. 数据仅供参考，不构成投资建议
